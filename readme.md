Aplikasi Manajemen Iuran Uang Kas

Deskripsi

Aplikasi Manajemen Iuran Uang Kas merupakan program yang dibuat menggunakan bahasa pemrograman Go. Program ini digunakan untuk membantu pengelolaan data pembayaran uang kas mahasiswa dalam satu kelas.

Melalui aplikasi ini, pengguna dapat memasukkan data pembayaran kas mahasiswa, melihat seluruh data yang tersimpan, mengecek status pembayaran berdasarkan NIM, melihat detail pembayaran mahasiswa, mengurutkan data, serta menghapus seluruh data apabila diperlukan.

Program ini dikembangkan sebagai implementasi materi yang dipelajari pada mata kuliah Algoritma Pemrograman, khususnya penggunaan array, function, searching, sorting, percabangan, dan perulangan.


Fitur

1. Input Uang Kas
2. Tampilkan Semua Data
3. Cek Status by NIM
4. Cek Detail by NIM
5. Urutkan Tampilan
6. Reset Data Kas
